# CareerRadar

CareerRadar is a job market analysis platform for students and junior developers.

The application helps users find junior/internship vacancies, analyze required skills, calculate job match scores, and identify personal skill gaps.

## MVP Features

- User profile
- User skills management
- Job vacancy aggregation
- Skill extraction from job descriptions
- Match score calculation
- Missing skills analysis
- Basic job market analytics

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Docker
- pytest

## Project Status

In development.

Build a practical backend project that demonstrates:

- REST API design
